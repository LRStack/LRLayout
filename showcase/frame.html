frame

A media box with a fixed aspect ratio that crops overflow. Drop in an image, video, picture, or iframe and it fills the frame — cropped via object-fit rather than distorted.

Default 16:9

A plain .lr-frame. The source image is 4:3, so the top and bottom are cropped to fill the wider frame.

Any ratio

Set --ratio to any aspect ratio. Same image, same object-fit: cover — only the frame's shape changes.

--ratio: 1 / 1
--ratio: 4 / 3
--ratio: 21 / 9

Cover vs contain

--object switches the fit. cover (default) crops to fill; contain fits the whole image inside, letterboxing against the frame's background.

object-fit: cover (default)
--object: contain
