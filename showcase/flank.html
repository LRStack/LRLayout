Form rows: aligned labels

Each row is a <label> that is itself a .lr-flank — the label text is the sidebar, the field fills the rest. The <form> is an .lr-stack. Setting --flank-basis once on the form sizes every row's label column to the same width, so they line up at the widest label. Unlike the per-instance --side, --flank-basis inherits, so a single declaration covers all the rows.

Narrow the window — each label wraps above its field.
